## Install

```
$ npm install --global ink-js-console
```

## Usage

```js
$ ink-js-console --help

  Usage
    ink-js-console [input]

  Options
    --name  Lorem ipsum [Default: false]

  Examples
    $ ink-js-console
    I love Ink
    $ ink-js-console --name=ponies
    I love ponies
```
